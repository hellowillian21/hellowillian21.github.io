Vuex的Action中参数解构的使用 
// 对象解构例子：
let { foo, bar } = { foo: "aaa", bar: "bbb" }
foo // "aaa"
bar // "bbb"

Action 函数接受一个与 store 实例具有相同方法和属性的 context 对象，因此你可以调用 context.commit 提交一个 mutation，或者通过 context.state 和 context.getters 来获取 state 和 getters。payload作为第二个参数（可选）

actions: {
    increment (context) {
      context.commit('increment')
    }
}

context对象包含以下属性：

{
    state,    // 等同于 store.state， 若在模块中则为局部状态
    rooState, // 等同于 store.state, 若存在于模块中
    commit,   // 等同于 store.commit
    dispatch, // 等同于 store.dispatch
    getters   // 等同于 store.getters 
}

可以理解为action中的函数会默认自动获取context这个对象为第一个参数。
而context这个对象拥有和store相同的属性和方法，从以上可以看到。
使用es6语法可以简写为: 

actions: {
  increment ({ commit }) {
    commit('increment')
  }
}

increment ({ commit }) { … }中的参数对象{..}就是指的context对象，而{ commit }就是指的context对象中的commit方法。
